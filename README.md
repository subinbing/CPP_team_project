### 2. 과제 개요
#### 2-1) 과제 배경 
요즘 대다수의 사이트에서 서비스를 이용하기 위해 회원가입 또는 로그인이 필수적이다. 개인마다 하나의 계정이 요구되는 현대의 디지털 시대에 맞게 그러한 로그인 시스템을 구현해보고자 함

#### 2-2) 과제 필요성 
C++을 이용한 MFC의 어플리케이션 제작 과정을 더욱 이해할 수 있고 활용할 수 있음

#### 2-3) 과제 선택 동기
